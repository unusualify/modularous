# Auth Traits

Two traits under `Auth/` extend the `User` model with registration verification and OAuth provider linking.

| Trait | Description |
|-------|-------------|
| [CanRegister](./can-register) | Email-verification token dispatch for the registration flow |
| [HasOauth](./has-oauth) | OAuth provider linking via `UserOauth` has-many |
